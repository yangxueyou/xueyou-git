# 别名

请在命令行输入如下东西，然后我们就可以使用快捷键了

```js
alias gc='git clone'
alias gco='git checkout'
alias gd='git diff'
alias gl='git pull'
alias gp='git push'
alias gr='git branch'
alias gst='git status'
alias glg='git log'
alias glgp='git log -p'
alias gad='git add .'
alias gcm='git commit -m'
```

```js
// 使用例子
$ gst
$ gco master
$ gl origin master
```
